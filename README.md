# 🚀 NexusHR - Enterprise HR Management System

NexusHR is a modern Enterprise Human Resource Management System (HRMS) built using **React.js**, **Spring Boot**, and **PostgreSQL**. It provides separate dashboards for Admin, Manager, and Employee with real-time workforce management features.

---

## 📌 Features

### 👨‍💼 Admin

* Employee Management
* Department Management
* Attendance Management
* Leave Approval
* Payroll Management
* Project Management
* Task Assignment
* Notification Management
* Reports & Analytics

### 👨‍💻 Manager

* Team Management
* Assign Tasks
* Leave Approval
* Project Monitoring
* Reports
* Notifications

### 👨‍💼 Employee

* Personal Dashboard
* Attendance
* My Tasks
* Leave Request
* Salary Details
* Notifications
* Profile Management

---

# 🛠 Tech Stack

### Frontend

* React.js
* Bootstrap 5
* Axios
* React Router

### Backend

* Spring Boot
* Spring Data JPA
* Spring Security
* REST APIs
* WebSocket

### Database

* PostgreSQL

### Deployment

* Railway (Backend)
* Netlify / Vercel (Frontend)

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/DevidasPund/NexusHR.git
```

## Frontend

```bash
cd frontend
npm install
npm start
```

## Backend

```bash
cd nexushr-backend
mvn spring-boot:run
```
